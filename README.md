# Customer-Segmentation-using-Unsupervised-ML
Customer Segmentation with Unsupervised ML: Uses K-Means, Hierarchical Clustering, &amp; DBSCAN to segment customers by Income, Purchases, &amp; Spending. Silhouette score of 0.6153 with 5 clusters, optimized to 3. Includes scatter plots, t-SNE, &amp; CRM integration for better engagement &amp; sales.
